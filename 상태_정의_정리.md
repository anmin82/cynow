# 용기 상태 정의 정리

## 표준 상태 목록

1. **보관** - 보관 중인 용기
2. **충전** - 충전 중/충전 완료된 용기
3. **분석** - 분석 중인 용기
4. **창입** - 창고 입고된 용기
5. **출하** - 출하된 용기
6. **이상** - 이상 상태 용기
7. **정비** - 정비 가능한 사용불가 용기 (정비 후 공용기로 사용 가능)
8. **폐기** - 실제 폐기된 용기 (삭제하지 않고 폐기 상태로 기록)

## 상태별 의미

### 정비 (950, 952)
- **의미**: 충전에 사용하지 못하는 용기
- **특징**: 정비를 하면 공용기로 사용 가능한 자원
- **가용 여부**: 비가용 (정비 후 사용 가능)
- **위험도**: 정비 상태 자체는 위험도에 직접 영향 없음

### 폐기 (990)
- **의미**: 정말 폐기된 용기
- **특징**: 삭제하지 않고 폐기 상태로 기록됨
- **가용 여부**: 비가용 (영구적으로 사용 불가)
- **위험도**: 폐기 수량이 있으면 HIGH 위험도

---

## 가용 수량 계산

- **가용**: 보관, 충전 상태만 포함
- **비가용**: 분석, 창입, 출하, 이상, 정비, 폐기

---

## 위험도 계산 기준

- **HIGH**: 
  - 가용 수량 = 0
  - 이상 수량 > 0
  - **폐기 수량 > 0** (실제 폐기된 용기)
  - 가용 수량 비율 < 10%
- **MEDIUM**: 가용 수량 비율 < 20%
- **LOW**: 가용 수량 비율 < 30%
- **NORMAL**: 그 외

---

## 주의사항

1. **정비 vs 폐기 구분**:
   - 정비(950, 952): 정비 후 사용 가능 → 자원으로 관리
   - 폐기(990): 영구적으로 사용 불가 → 폐기 처리 필요

2. **정비 용기는 가용 수량에 포함되지 않음**:
   - 정비 후 공용기로 전환되면 보관/충전 상태로 변경됨

3. **폐기 용기는 위험도에 영향**:
   - 폐기 수량이 있으면 HIGH 위험도로 표시
   - 처리 계획이 필요함
